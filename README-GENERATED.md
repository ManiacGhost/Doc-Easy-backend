# Doc-Easy Backend Documentation

---

## 1. Setup

### Prerequisites
- Java 17+
- Maven 3.8+
- Docker & Docker Compose
- PostgreSQL (if not using Docker)
- Redis (if not using Docker)

### Environment Variables
Configure your `.env` file or set environment variables for:
- DB_USERNAME, DB_PASSWORD, DB_URL
- JWT_SECRET, JWT_EXPIRATION
- MAIL_HOST, MAIL_PORT, MAIL_USERNAME, MAIL_PASSWORD, MAIL_FROM
- AWS/Cloudinary/AI keys as needed

### Local Setup
```bash
# Clone repository
 git clone <repo-url>
 cd Doc-Easy-backend

# Start all services (app, postgres, redis)
docker compose up -d --build
```

## 2. Running Instructions

- To run locally (without Docker):
  1. Start PostgreSQL and Redis manually
  2. Configure `src/main/resources/application.yml`
  3. Run: `./mvnw spring-boot:run`

- To run with Docker Compose:
  ```bash
  docker compose up -d --build
  ```
- To stop:
  ```bash
  docker compose down
  ```

## 3. API Documentation

### Authentication
- `POST /api/auth/register` — Register user
- `POST /api/auth/login` — Login, returns JWT
- `POST /api/auth/verify-otp` — Verify OTP

### Asset Management
- `POST /api/assets/upload` — Upload asset
- `GET /api/assets/{id}` — Get asset info
- `GET /api/assets` — List all assets

### Q&A
- `POST /api/ask` — Ask a question about an asset
- `GET /api/qa/{id}` — Get Q&A record

### Common Response Format
```json
{
  "status": 200,
  "data": { ... },
  "error": null
}
```

### Error Handling
- 401: Invalid credentials
- 409: Email already registered
- 400: Validation errors
- 404: Not found
